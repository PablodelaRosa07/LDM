Cómo Dominar Flexbox en 10 Minutos

Publicado el 5 de Dic de 2025 por **Ana M.** en **Desarrollo Web**

CSS Flexbox es, sin duda, una de las herramientas más importantes para el diseño web moderno. Nos permite distribuir, alinear y ordenar el espacio entre los ítems de un contenedor de una manera predecible y sencilla.

Conceptos Clave

Todo en Flexbox se basa en dos ejes: el **eje principal (main axis)** y el **eje cruzado (cross axis)**.

La propiedad más importante para el contenedor es `display: flex;`.

.contenedor {
    display: flex;
    justify-content: center; /* Centra horizontalmente */
    align-items: center; /* Centra verticalmente */
}

¡Con solo esas líneas de código, hemos resuelto muchos dolores de cabeza de alineación!

¿Por qué usarlo?

  • **Alineación Sencilla:** Olvídate de los `margin: auto;` complicados.
  • **Distribución de Espacio:** Controla el espacio entre ítems (`space-between`, `space-around`).
  • **Responsividad:** Es la base para diseños que se adaptan a cualquier pantalla.
